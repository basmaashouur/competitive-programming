## String Functions
```cpp
string s; int i; char c;

// Convert a string to int
i = stoi(s); // "97" = 97

// Convert an int to string
s = to_string(i); // 97 = "97"

// Convert char to int
i = x - '0';  // '1' = 1

// Convert int to char
c = x + '0'; // 1 = '1'

// Cast a char, read the value as an ascii code
s = int(c); // 'a' = 97

// Cast an int, read the ascii code to char
c = char(i); 97 = 'a'
```
